## 3. 主要差距（按严重级）

### CRITICAL

1. 缺少鉴权与 RBAC，`/api/admin/*` 可直接调用（存在高风险运维接口暴露）。
2. MinerU 处理链仍为 stub，无法处理真实扫描 PDF 与文档后处理要求。

### HIGH

1. Hybrid 检索关系过滤在 API/Chat 实际路径基本失效（空实体索引）。
2. 实体 ID 生成不稳定（`hash(name)`），不符合稳定 ID/Hash 过滤方案。
3. LLM 路由未实现 tier failover/circuit breaker，策略配置未落地执行。
4. 外发 LLM 调用前未做默认脱敏，且 QA 生成未注入证据上下文。
5. 上传流程无文件大小/类型限制，且整文件读入内存。
6. 关键表格结构化、页类型识别、关系构建未形成完整流水线。

### MEDIUM

1. 评测链路未形成 manifest -> runner -> MinIO/DB -> 管理端闭环。
2. 部署拓扑缺少 scheduler（资质临期巡检、评测调度）。
3. 前端仍是演示态，未完成检索/问答/抽查/评测闭环联动。
4. `CONFIG_DIR` 在运行时传入但配置加载器未按环境目录读取。

### LOW

1. Docker/示例环境默认口令存在误用风险，需 secrets 化。
